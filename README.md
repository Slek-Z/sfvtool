# SFVTool
A C++ open-source, multi-platform Simple File Verification (SFV) tool

## Dependencies

* [CMake](https://cmake.org/)

* [Boost](https://www.boost.org/)

## Usage
```
Usage: sfvtool [OPTION]... [FILE]...
Print or check Simple File Verification (SFV) checksums.

Options:
  -c, --check           read SFVs from FILEs and check them
      --ignore_missing  don't fail or report status for missing files (with check)
      --quiet           don't print OK for each successfully verified file (with check)
      --status          don't output anything, status code shows success (with check)
      --strict          exit non-zero for improperly formatted checksum lines (with check)
  -w, --warn            warn about improperly formatted checksum lines (with check)

      --help            display this help and exit
      --version         output version information and exit
```
